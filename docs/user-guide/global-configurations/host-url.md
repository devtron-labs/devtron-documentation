# Host URL

Host URL is the domain address at which your devtron dashboard can be reached.  

### Add Host URL

Navigate to **Global Configurations** â†’ **Host URL**.

![](https://devtron-public-asset.s3.us-east-2.amazonaws.com/images/devtron-v2/global-configurations/host-url/host-url.jpg)
<center>Figure 1: Host URL</center>

On the **Host URL** page:

* Enter the host URL in the `Host URL` field. 
* Or, you can select auto-detect from your browser.
* Next, click `Update`.
