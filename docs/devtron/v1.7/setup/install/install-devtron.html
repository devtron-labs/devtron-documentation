Get Admin Credentials

Note

Refer Install Devtron to know the available tiers and installation options.

Using Own Kubernetes Cluster?â€‹

When Devtron is installed on your own Kubernetes cluster, it creates a default admin user and password (with unrestricted access to Devtron). You can use that credentials to log in as an administrator.

Username: admin
Password: Run the following command to get the admin password:

kubectl -n devtroncd get secret devtron-secret \
-o jsonpath='{.data.ADMIN_PASSWORD}' | base64 -d

Using Devtron Cloud (SaaS)?â€‹

If you are using the 30-day trial version of Devtron Cloud (SaaS), follow the steps below to get the credentials.

  1. Go to Devtron's License Dashboard and sign in using SSO or registered email address used at the time of installation.

  2. Once logged in, the Devtron License Dashboard will show your existing license. Below the license, you will find the Dashboard URL and login password (username will be admin).
